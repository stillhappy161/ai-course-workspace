## 贡献规范

1. 每条素材末尾标注 `[贡献者: xxx] [日期: yyyy-mm-dd] [来源: URL/书名/播客名]`
2. 修改已有素材时，在原贡献者后追加 `[修改: xxx] [日期: yyyy-mm-dd]`
3. 矛盾观点不要删除，标注 `⚡ 观点冲突：见[xxx]`
4. 状态标签统一使用：`#可入课` / `#待验证` / `#仅参考` / `#已过期` / `#备选` / `#已使用`
