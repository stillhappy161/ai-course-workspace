# 课程模块生成快捷指令

## 用法
在 VS Code 对话框中输入：**"生成模块X的内容"**

## 智能体行为
1. 扫描知识库中与该模块相关的所有素材
2. 检查素材缺口，主动建议联网搜索补充
3. 按 module-template.md 格式生成内容
4. 每页PPT包含：标题 + 核心信息 + 支撑素材 + 视觉建议 + 演讲备注
5. 标注草稿状态，等待用户审核

## 前置条件
- course-structure.md 中该模块的大纲已确认
- 知识库中有足够的素材支撑

## 输出位置
course-output/ 目录下新建 `module-X-content.md`

---

## 示例

**输入**：
> 生成模块1（认知篇）的内容

**智能体行为**：
1. 扫描 frameworks.md、analogies.md、stories-quotes.md、stats-data.md
2. 发现缺少"2026年大模型能力现状"的最新数据 → 建议联网搜索
3. 生成 M1 的完整内容，包含：
   - 开场设计（电气化类比 + 开场自测互动）
   - 4个小节的PPT内容（每节3-4页PPT）
   - 1个互动环节（AI程度自测）
   - 收尾和过渡
