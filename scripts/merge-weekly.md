# 每周合并汇总快捷指令

## 用法
在 VS Code 对话框中输入：**"本周汇总"**

## 智能体行为
1. 扫描本周三人各自的贡献（按 CONTRIBUTORS.md 的标识符）
2. 去重（同一来源、同一素材只保留一份）
3. 按分类整理本周新增内容
4. 标注：哪些观点一致（可以确认）、哪些观点矛盾（值得讨论）
5. 对照课程大纲，标注素材缺口
6. 输出本周合并报告

## 输出格式

```
📊 本周合并报告 | Week X（日期范围）

👥 贡献统计
- 你：X条（框架/痛点/类比）
- 合伙人A：X条（技术/工具/风险）
- 研究生B：X条（案例/数据/竞品）

📦 本周新增素材（按分类）
- 案例库：+X条
- 框架库：+X条
- 数据池：+X条
- ...

⚡ 观点碰撞（值得深入讨论）
1. [矛盾点描述]
2. ...

🔍 课程素材缺口
- [缺口1]：需要什么 / 紧急程度
- [缺口2]：...

✅ 下周建议任务
- 你：[建议]
- 合伙人A：[建议]
- 研究生B：[建议]
```
