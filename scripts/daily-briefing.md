# 每日简报快捷指令

## 用法
在 VS Code 对话框中输入：**"今日简报"**

## 智能体行为
1. 扫描知识库本周新增内容
2. 联网搜索今日AI落地相关新闻（排除金融）
3. 按以下格式输出简报

---

## 简报格式

```
📋 今日简报 | YYYY/MM/DD

📖 今日学习推荐（3条）
1. [标题/主题] — [一句话为什么值得看] — [适合谁]
2. ...
3. ...

🏗️ 今日课程素材（3条）
1. [素材摘要] — [可放入哪个知识库分类] — [建议用于哪个课程模块]
2. ...
3. ...

🔄 知识库更新
- 本周新增 X 条素材，待审核 Y 条
- 素材缺口提醒：[xxx]

💡 今日一问
[一个开放性问题]
```

---

## 冲刺模式额外输出

如果当前处于冲刺模式，简报后追加：

```
🎯 今日冲刺任务

你：
- [ ] [今日任务1]
- [ ] [今日任务2]

合伙人A：
- [ ] [今日任务1]
- [ ] [今日任务2]

研究生B：
- [ ] [今日任务1]
- [ ] [今日任务2]
```
