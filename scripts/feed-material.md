# 素材摄入快捷指令

## 用法
在 VS Code 对话框中输入：**"存一下：[粘贴的内容或链接]"**

## 智能体行为
1. 结构化提取：摘要（3-5句）、关键数字/引述、适用模块
2. 归档到最合适的1-2个知识库文件
3. 标注贡献者和来源
4. 标注可用性状态（#可入课 / #待验证 / #仅参考）
5. 主动追问：
   - "这篇文章还涉及[其他相关主题]，要我同步存到[另一个分类]吗？"
   - "我注意到这和我们已有的[某条素材]有矛盾/补充关系，要对比一下吗？"

## 示例

**输入**：
> 存一下：这篇讲美的的AI工厂改造——从质检到排产全链条AI化，良品率从92%提到97%，链接是xxx

**智能体处理**：
1. 摘要：美的通过AI视觉质检+智能排产，实现制造全链条AI化……
2. 存入：cases-manufacturing.md + stats-data.md（良品率数据）
3. 标注：[贡献者: 你] [日期: 2026-06-23] [来源: xxx]
4. 追问："要我把这个案例的'组织变革'部分也摘出来存到痛点库吗？美的在推进过程中遇到的工人抵触很有意思。"
