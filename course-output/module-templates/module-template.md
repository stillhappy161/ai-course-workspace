# 模块模板

> 每个课程模块按此模板填充内容。生成PPT时，每个小节 = 2-4页PPT。

---

## 模块信息

- **模块编号**：M[X]
- **模块名称**：[名称]
- **时长**：[X]分钟
- **核心问题**：[这个模块回答什么问题]

---

## 模块结构

### 开场（[X]分钟）
- 钩子：一个故事/数据/问题
- 目标预告：接下来X分钟你会得到什么

### 小节1：[标题]（[X]分钟）
- **核心信息**（3-5 bullet points）
- **支撑素材**：案例/数据/类比（来自知识库哪个文件）
- **PPT页数**：预计X页
- **演讲备注**：讲师提示

### 小节2：[标题]（[X]分钟）
- **核心信息**
- **支撑素材**
- **PPT页数**
- **演讲备注**

### 小节3：[标题]（[X]分钟）
- **核心信息**
- **支撑素材**
- **PPT页数**
- **演讲备注**

### 互动环节（[X]分钟）
- **类型**：讨论/投票/沙盘/自测/角色扮演
- **问题/场景**
- **分组建议**
- **引导提示**
- **预期结论**

### 收尾（[X]分钟）
- **本模块金句**
- **过渡到下一模块的桥接**

---

## PPT内容生成格式

每页PPT按以下格式生成：

```
【PPT页 X/Y】
标题：xxx
类型：标题页 / 信息页 / 案例页 / 数据页 / 互动页 / 过渡页
核心内容（3-5 bullet points，每点不超过20字）：
• xxx
• xxx
视觉建议（描述图表/图片/图标）：
演讲备注（讲师说的话）：
素材来源（引用知识库哪条）：
```
